<h1>프로젝트</h1>

<h2>CinemaInfo</h2>
  <p>TMDB API를 활용한 고성능 SPA영화 정보 제공 사이트 </p>
  <h2>기능</h2>
  <ul>
     <li>🎬전체영화 및 모든 영화의 제목,장르,배우등의 정보 제공</li>
     <li>>🔍검색어 입력시 키워드에 맞는 모든영화 정보 제공</li>
     <li>⭐️영화의 대한 리뷰 게시판을 통해 리뷰작성및 수정,삭제,별점 구현</li>
     <li>Route를 이용한 SPA구현</li>
  </ul>
